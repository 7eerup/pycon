## Python
```bash
변수란: 값을 저장하고 재사용하기 위한 도구
메서드(Method): 변수에 저장되는 데이터

재사용·변경 용이: 한 곳을 바꾸면 계산 전체가 자동 반영
의미 부여로 가독성 향상: magic number 대신 이름으로 의도 전달
중복 제거·실수 감소: 같은 값을 여러 번 쓰지 않음
중간 결과 보관: 복잡한 계산을 단계로 나눔
유연성 확보: 입력(데이터)에 따라 동작이 달라짐
유지보수·디버깅 쉬움: 추적과 수정이 간단
```

### 기본 자료형
int, str, bool: 하나의 값만 저장
list: 여러 값을 순서대로 저장
dict: 여러 값을 이름(키)으로 저장


### for 활용
- 리스트, 딕셔너리 순회
- 반복 횟수가 정해져 있음
- 1부터 10까지 같은 범위 반복

### while 활용
- 사용자 입력을 받을 때
- 조건이 언제 끝날지 모를 때
- 특정 조건이 만족될 때까지 반복
