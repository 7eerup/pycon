# Mini NPU Simulator

## 프로젝트 개요
본 프로젝트는 N×N 크기의 입력 패턴과 필터 간의 MAC(Multiply-Accumulate) 연산을 통해  
패턴을 분류하는 간단한 NPU(Neural Processing Unit) 시뮬레이터를 구현하는 것을 목표로 한다.

외부 라이브러리를 사용하지 않고 Python 표준 라이브러리만을 활용하여 구현하였다.

---

## 주요 기능

### 1. 사용자 입력 모드 (3×3)
- 3×3 필터 2개(A, B) 입력
- 3×3 패턴 입력
- MAC 연산 수행 후 점수 비교
- 결과 판정: A / B / UNDECIDED

---

### 2. data.json 분석 모드
- 다양한 크기(5×5, 13×13, 25×25)의 필터 및 패턴 로드
- 패턴별 MAC 연산 수행
- expected 값과 비교하여 PASS / FAIL 출력
- 전체 결과 요약 제공

---

### 3. 성능 분석
- 크기별 MAC 연산 시간 측정 (평균 10회)
- 출력 항목:
  - 크기 (N×N)
  - 평균 실행 시간(ms)
  - 연산 횟수(N²)

---

## 핵심 구현 로직

### MAC 연산
모든 연산은 반복문을 통해 직접 구현하였다.

---

### MAC(Multiply-Accumulate) 연산
- Multiply-Accumulate = 각 위치의 값을 곱한 뒤 모두 더하는 연산


### AI에서 MAC이 중요한 이유
- 신경망(Neural Network)
- CNN(Convolutional Neural Network)
- 행렬 곱 연산

이유:
- 뉴런의 출력 계산 = 입력 × 가중치 + 합
- 필터(가중치)와 입력 데이터의 유사도를 계산

---

## 유사도 계산 원리

입력 패턴과 필터를 같은 위치끼리 곱하고 모두 더하면:

- 값이 클수록 → 패턴이 필터와 유사
- 값이 작을수록 → 유사하지 않음
결과:
- 같은 위치에서 1이 겹치면 점수 증가
- 패턴 형태를 “얼마나 닮았는지” 측정 가능
